# 📈 AI Equity Research Agent

Real-time stock analysis powered by Claude AI with live web search.

## Features
- Real-time data via Anthropic web_search tool
- Streaming analysis with live progress
- Fundamental, Technical & Sentiment analysis
- 3-month price forecast with confidence
- Multi-stock comparison (AAPL, NVDA, TSLA, INTC)
- Apple Design System UI (dark mode)
- Mobile-responsive layout

## Usage
1. Open `index.html` in a browser
2. Enter your Anthropic API key (stored in localStorage)
3. Select a stock or "Compare All"
4. Click "Анализировать рынок"

## API Key
Get your key at https://console.anthropic.com/
The key is stored locally in your browser — never sent anywhere except Anthropic's API.

## Hosting
Deploy `index.html` to any static host: Netlify, Vercel, GitHub Pages, nginx.
No backend or build step required.
